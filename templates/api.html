API Documentation

Introduction

This documentation provides information on how to use the URL shortener API.

Endpoints

/api/shorten

Shortens a URL using the API.

        POST /api/shorten
        Headers:
            Authorization: (Header) {api_token}
        Body:
            {
                "url": "http://s1d.me/",
                "length": 6,
                "allow_numbers": true,
                "allow_special": false,
                "allow_uppercase": true,
                "allow_lowercase": true,
                "expiry_time": "2023-12-31 23:59:59"
            }
        Response:
            {
                "short_code": "abc123"
            }

/api/generate_token

Generates a new API token.

        POST /api/generate_token
        Response:
            {
                "token": "generated_token"
            }

/api/analytics

Gets analytics for the API token.

        GET /api/analytics
        Headers:
            Authorization: Bearer {api_token}
        Response:
            {
                "link_count": 10,
                "click_count": 50,
                "links": [
                    {
                        "short_code": "abc123",
                        "original_url": "http://s1d.me/",
                        "click_count": 5,
                        "expiry_time": "2023-12-31 23:59:59"
                    },
                    ...
                ]
            }

Rate Limiting

The API endpoints have rate limits to prevent abuse:

  • /shorten: 10 requests per minute
  • /api/shorten: 50 requests per minute
